# Milestone 4 — پنل مدیریت منو

هدف: مدیر بتواند دسته‌ها و محصولات را از پنل لپ‌تاپ مدیریت کند (افزودن/ویرایش/حذف/قیمت/موجودی) و تغییرات بلافاصله در منوی مشتری دیده شود.

## صفحه مدیریت (AdminPage)
- [x] مسیر `/admin` پشت همان لاگین (گارد توکن از M3)
- [x] چیدمان دو بخشی: مدیریت دسته‌ها + مدیریت محصولات

## مدیریت دسته‌ها
- [x] لیست همه دسته‌ها (حتی غیرفعال) از `GET /api/admin/categories`
- [x] افزودن دسته جدید (نام + ترتیب نمایش)
- [x] ویرایش inline نام و ترتیب
- [x] سوییچ فعال/غیرفعال کردن دسته (غیرفعال = حذف از منوی مشتری)
- [x] حذف دسته با تاییدیه — و پیام خطای واضح اگر محصول دارد

## مدیریت محصولات
- [x] جدول همه محصولات (حتی ناموجود) با فیلتر بر اساس دسته
- [x] افزودن محصول: نام، دسته، قیمت (تومان)، توضیح، ترتیب
- [x] **ویرایش سریع قیمت**: کلیک روی قیمت → ورودی عددی → ذخیره با Enter (مهم‌ترین کار روزمره مدیر)
- [x] سوییچ موجود/ناموجود با یک کلیک
- [x] ویرایش کامل محصول در مودال
- [x] حذف محصول با تاییدیه (اگر در سفارشی بوده → ناموجود می‌شود، با توضیح به مدیر)
- [x] اعتبارسنجی فرم‌ها: قیمت عدد صحیح مثبت، نام غیرخالی — با پیام فارسی

## اعمال فوری روی منوی مشتری
- [x] تأیید مسیر کامل: تغییر در پنل → ذخیره در دیتابیس → منوی مشتری با refetch (لود صفحه یا چرخه ۳۰ ثانیه‌ای) قیمت جدید را نشان می‌دهد
- [x] تست: قیمت را عوض کن، در موبایل بدون رفرش دستی ظرف ۳۰ ثانیه قیمت جدید بیاید

## جمع‌بندی نهایی پروژه

- [x] README اصلی پروژه: راه‌اندازی کامل backend + frontend از صفر، ساخت QR منوی چاپی (URL صفحه منو)، نکات شبکه local
- [x] تست end-to-end کامل هر ۴ milestone پشت سر هم (سناریوی یک روز کاری واقعی کافه)
